◆ addTurmaStudent()

bool GestaoHor::addTurmaStudent ( int  n,
const UCTurma &  turma 
)

Searches for the student with code n and tries to add the desired class to their turmas field, calling Student::addTurma(), and increments the size (occupation) of the class, calling UCTurma::incrementSize().
Returns false if the provided student code does not belong to any student, the same for the class, if the change unbalances the classes' occupation in the desired UC or if the student's schedule is not compatible with the change, true otherwise.
Complexity: O(n²), being n the number of classes the student is enrolled in.
